# Prisma

Thu muc nay chi giu cac thanh phan Prisma dang duoc su dung:

- `schema.prisma`
- `migrations/`
- `seed.js`

## Yeu cau

- Database provider: `mysql`
- Env bat buoc: `DATABASE_URL`

## Lenh hay dung

- `npm run prisma:generate`
- `npm run prisma:migrate`
- `npm run prisma:studio`
- `npm run prisma:seed`
